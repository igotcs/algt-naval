Antes de começarmos o projeto, buscamos inspirações em projetos de outros estudantes no github e outras redes sociais, pra ter ao menos uma referencia de como começar e compreender as melhores práticas para a manipulação de matrizes e sistemas de coordenadas. A priori, pensamos em usar headers para separar a lógica em diferentes arquivos locais. Mas, devido a limitações técnicas optamos por fazer um código único, onde o único arquivo externo é o data.txt, que armazena dados como número da rodada, número de acertos, erros e também as informações dos tabuleiros de cada jogador.

A primeira parte do codigo foi fazer as funcoes que iriam ser usadas ao longo do jogo, como a de atacar, imprimir tabuleiro e posicionar os navios.

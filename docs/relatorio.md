RELÁTORIO
	Neste trabalho usamos as bibliotecas stdio.h, string.h, e ctype.h, esta última para uma única função chamada toupper, que converte variáveis do tipo char que estão minúsculas para a sua forma maiúscula. Também incluímos a biblioteca stdlib.h, mas esta não teve uso.
  Primeiramente pensamos em fazer a parte inicial especificada no roteiro, que é o menu. Então fizemos uma função menu que apenas imprimia o menu com as opções de o que o usuário poderia fazer. Após ler a resposta do usuário idealizamos fazer um switch case, mas logo a ideia foi descartada por que era necessário fazer um loop do menu e leitura de resposta caso a resposta não fosse uma das esperadas, então usamos while.
